# Interfaz

App de votaciones en tiempo real con React Native y Expo.

## Stack

- React Native 0.81.4
- Expo ~54.0.10
- TypeScript ~5.9.2
- React Native Paper 5.14.5
- React Navigation

## Instalación

```bash
npm install
```

## Ejecución

```bash
npm start        # Servidor de desarrollo
npm run android  # Android
```

## Plataforma

- Android
